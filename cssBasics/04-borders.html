Borders

The border property allows you to specify the style, width, and color of an element's border.

Border Style

The border-style property can take up to four values (top right bottm and left border>.

The following border styles are allowed:
  • Dotted
  • Dashed
  • Solid
  • Double
  • Groove: 3d, depends on border color values
  • Ridge: 3d, " "
  • Inset: 3d " "
  • None: none
  • Hidden
  • Mixed: you can set a different border style for the top left right and bottom
