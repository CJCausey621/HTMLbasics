Classes

A CSS classs is an identifier of an element which can be used multiple times on a page

Classes in schools are periods, so remember to use a period . when you want to reference a class in CSS

  • Classes start with a . in the selector the (.anythong-you-want-to-name-it)
  • The same class name can be used more than once in the same file
  • The class keyword refers to the class name that you set in the HTML
