CSS IDs

An ID is a !unique! identifier

ID notes

  • In CSS, an ID starts with an Octothorpe
  • Example use for IDs: Targeting a specific header or section
  • Each ID should only be used once per page!

    • This is important with JavaScript as well
  • Use it for one off styling.

IDs versus Classes

IDs

This ID will be linked in a CSS page by a #

You cant have two ids on an element, they will cancel each other out

Classes

Elements can have any number of classes

For example, this element has two classes

You CAN use an ID twice, but you should not.
